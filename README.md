#  Book Management System

A React-based CRUD application for managing books, integrated with MockAPI.

## Live Demo
🔗 https://book-management-nine-sooty.vercel.app

## GitHub Repository
🔗 https://github.com/Abhishek652004/book-management

## Features
- View all books in a responsive grid
- Add new books via a form
- Edit existing book details
- Delete books with confirmation
- Search books by title or author
- Filter books by genre
- Loading and error state handling

## Tech Stack
- React.js
- Axios (API calls)
- MockAPI (backend)
- Vercel (deployment)

## Setup Instructions

### 1. Clone the repository
git clone https://github.com/Abhishek652004/book-management.git
cd book-management

### 2. Install dependencies
npm install

### 3. Run the app
npm start

The app will open at http://localhost:3000
